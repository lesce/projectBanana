The user

  • Can connect to the game using his Facebook account
    • He can invite 2-3 friend to join a game
    • He can create a game and wait for 2-3 players to join the game

The rules/gameplay

  • When the game starts the map and the players order are generated automaticaly
  • Each player chooses a location for 2 villages and a road for each village
  • The resources from the last village are automaticaly added to his inventory
  • At each dice rolling the resources are added automaticaly to each user
  • The user can select a node to build a village or a village to build a town
    the costs of the action will be extracted automaticaly from his inventory
  • The user can select a path to build a road , the cost of the road will be extracted
    automaticaly from the users inventory
  • A village can be built only if its at a distance of minimum 2 paths from other villages
  • A path can be built only if it's linked to another path or another village/town
  • The user can buy special cards and use them the next round
  • The special cards can't be seen from other players until they are being used
  • If somebody rolls a 7 each player that has more than 7 cards will lose half his cards
    the player who threw the dice has to block a hexagon and pick a card from one of the players
  • The user can offer to exchange one or more resources
  • The game ends when a player reaches 10 points

Points

  • Village = 1 point
  • Town = 2 points
  • The longest road ( >= 5 ) = 2 points
  • The biggest army ( >= 3 soldiers ) = 2 points
  • Special cards = 1 point each

Special cards

  • Soldier - it's the same as throwing a 7 (it doesn't matter how many cards a player is holding)
  • 2 resources - the player can choose 2 resources from the "bank"
  • 2 roads - the player can build 2 road for free
  • 1 point - the player has a hidden point
  • monopol - the player can choose a resource that he wants to take from the other players
